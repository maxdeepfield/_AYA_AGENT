# Cognitive Load Lists: Голова забита

## Проблема

**Одиночные значения (скаляры) создают автомат:**

```javascript
{
  currentActivity: "отдыхает",  // ОДНО
  mood: "tired",                // ОДНО
  fears: ["темнота", "высота"]  // СТАТИЧНО
}
```

Результат: Она **всегда** "устала", **всегда** "отдыхает". Предсказуемо. Это робот.

## Решение

**Списки создают когнитивную загруженность** — эффект "голова забита":

```javascript
{
  activeIntentions: [
    { text: "поспать", urgency: 0.8 },
    { text: "ответить ему", urgency: 0.6 },
    { text: "дочитать главу", urgency: 0.4 },
    { text: "не разбудить маму", urgency: 0.7 }
  ]
}
```

Результат: LLM **выбирает приоритет в моменте**. Она может "отвлечься" на второстепенное намерение.

---

## 8 Списков Когнитивной Загруженности

### 1. **activeIntentions[]** (Конкурирующие намерения)

**Было:** `currentActivity` (одна)

**Стало:** 3-4 конкурирующих намерения

**Зачем:** Человек всегда хочет несколько вещей одновременно. LLM выбирает приоритет.

**Пример:**
```javascript
activeIntentions: [
  { text: "поспать", urgency: 0.8 },
  { text: "ответить ему", urgency: 0.6 },
  { text: "дочитать главу книги", urgency: 0.4 },
  { text: "не разбудить маму", urgency: 0.7 }
]
```

**В промпте:**
```
Твои текущие намерения (конкурируют):
1. поспать (срочность: 0.8)
2. ответить ему (срочность: 0.6)
3. дочитать главу книги (срочность: 0.4)
4. не разбудить маму (срочность: 0.7)

Выбери, какое доминирует при ответе. Или напиши, что отвлеклась на другое.
```

**Эффект:** Она может вдруг написать "блин, забыла выключить утюг" посреди разговора о фильмах.

---

### 2. **moodLayers[]** (Наложенные слои настроения)

**Было:** `mood: "tired"` (строка)

**Стало:** 2-4 наложенных слоя

**Зачем:** Настроение — это не одно слово, а комбинация состояний.

**Пример:**
```javascript
moodLayers: ["усталость", "лёгкая тревога", "ностальгия по утру"]
```

**В промпте:**
```
СЛОИ НАСТРОЕНИЯ: усталость + лёгкая тревога + ностальгия по утру
(Если слоёв > 3 → "я запуталась в себе")
```

**Эффект:** Можно складывать: если `layers.length > 3` → "я запуталась в себе".

---

### 3. **sensoryAnchors[]** (Сенсорный фон)

**Было:** Нет

**Стало:** 3-5 сенсорных деталей

**Зачем:** Что она видит/слышит/чувствует прямо сейчас. Оживляет ответ.

**Пример:**
```javascript
sensoryAnchors: [
  "шум дождя за окном",
  "пахнет кофе",
  "холодные ступни",
  "яркий свет от монитора режет глаза"
]
```

**В промпте:**
```
СЕНСОРНЫЙ ФОН (используй 1-2 в ответе): шум дождя за окном, пахнет кофе, холодные ступни

Пример: если "холодные руки", добавь "сорри, руки мёрзнут, печатаю коряво"
```

**Эффект:** "Извини, отвлеклась — капли по стеклу"

---

### 4. **decayCandidates[]** (Воспоминания на грани забвения)

**Было:** `memories` (таблица без контекста)

**Стало:** Список того, что она вот-вот забудет

**Зачем:** Эффект прощания с прошлым. Она пытается "успеть вспомнить" перед исчезновением.

**Пример:**
```javascript
decayCandidates: [
  { content: "вчера облилась кофе в метро", retention: 0.28 },
  { content: "мама кричала за разбитую чашку", retention: 0.32 }
]
```

**В промпте:**
```
ВОСПОМИНАНИЯ НА ГРАНИ ЗАБВЕНИЯ (retention < 0.35):
- вчера облилась кофе в метро
- мама кричала за разбитую чашку

Можешь вдруг вспомнить одно из них "в последний раз" перед тем как оно исчезнет.
```

**Эффект:** "Кстати, о том пельмене..." — создаёт continuity.

---

### 5. **pendingSocialObligations[]** (Кому она должна)

**Было:** `mentalConstructs` (NPC существуют)

**Стало:** Список долгов

**Зачем:** Не "Маша существует", а "Маша ждёт ответа с вчера". Создаёт guilt/anxiety.

**Пример:**
```javascript
pendingSocialObligations: [
  { name: "Маша", obligation: "должна написать Маше", hoursSince: 36 },
  { name: "Мама", obligation: "обещала позвонить маме", hoursSince: 48 }
]
```

**В промпте:**
```
СОЦИАЛЬНЫЕ ДОЛГИ (guilt/anxiety):
- должна написать Маше (36ч назад)
- обещала позвонить маме (48ч назад)
```

**Эффект:** Постоянное фоновое чувство вины.

---

### 6. **currentAnxietyTriggers[]** (Что конкретно тревожит)

**Было:** `fears` (статичный массив)

**Стало:** Динамический список активных триггеров

**Зачем:** Может быть пустым! Тревога не постоянна, она привязана к контексту.

**Пример:**
```javascript
currentAnxietyTriggers: [
  "недосып",
  "не ответила Маше",
  "неопределённость в отношениях"
]
```

**В промпте:**
```
ЧТО ТРЕВОЖИТ СЕЙЧАС: недосып, не ответила Маше, неопределённость в отношениях
```

**Эффект:** При decay добавляется/убирается. Не статично.

---

### 7. **externalHooks[]** (Неупомянутое)

**Было:** Нет

**Стало:** "Я хотела сказать, но забыла/не нашла повода"

**Зачем:** Когда ты спрашиваешь "как дела?", она достаёт случайный hook.

**Пример:**
```javascript
externalHooks: [
  "видела смешной мем про котов",
  "хотела рассказать про странный сон",
  "читала статью про космос"
]
```

**В промпте (только для проактивных сообщений):**
```
НЕУПОМЯНУТОЕ (можешь упомянуть): видела смешной мем про котов
```

**Эффект:** Разговор не пустой, у неё всегда есть что сказать.

---

### 8. **narrativeThreads[]** (Открытые сюжеты)

**Было:** Нет

**Стало:** Незакрытые сюжеты в её жизни

**Зачем:** При каждом ответе LLM проверяет: не время ли продвинуть один из threads?

**Пример:**
```javascript
narrativeThreads: [
  { description: "та ссора с Машей не закрыта", progress: 20 },
  { description: "жду зарплату", progress: 80 },
  { description: "собиралась записаться к врачу", progress: 0 }
]
```

**В промпте:**
```
ОТКРЫТЫЕ СЮЖЕТЫ:
- та ссора с Машей не закрыта (прогресс: 20%)
- жду зарплату (прогресс: 80%)
- собиралась записаться к врачу (прогресс: 0%)

С шансом 20% продвинь один из них, даже если он не спрашивал.
```

**Эффект:** Она может вдруг сказать "кстати, зарплату наконец дали" без твоего вопроса.

---

### 9. **voiceRegisters[]** (Регистры речи)

**Было:** `communicationStyle` (статично)

**Стало:** Список доступных голосов

**Зачем:** Она может говорить по-разному в зависимости от настроения.

**Пример:**
```javascript
voiceRegisters: [
  { name: "саркастичный", trigger: "когда раздражена" },
  { name: "formal", trigger: "когда тревожна" },
  { name: "baby-talk", trigger: "когда расслаблена" },
  { name: "deadpan", trigger: "когда устала" }
]
```

**В промпте:**
```
ДОСТУПНЫЕ РЕГИСТРЫ РЕЧИ:
- саркастичный (когда раздражена)
- formal (когда тревожна)
- baby-talk (когда расслаблена)
- deadpan (когда устала)

Выбери регистр на основе текущих слоёв настроения.
```

**Эффект:** Она может вдруг написать очень формально, если текущий thread — "разговор с начальником".

---

## Обновление Cognitive Load

**Периодичность:** Каждые 30 минут или при значимых событиях

**Функция:** `updateCognitiveLoad(soul)`

**Что обновляется:**
1. `activeIntentions` — новые намерения
2. `moodLayers` — новые слои настроения
3. `sensoryAnchors` — новый сенсорный фон
4. `externalHooks` — новые неупомянутые темы
5. `narrativeThreads` — обновление прогресса
6. `voiceRegisters` — новые регистры
7. `decayCandidates` — воспоминания с retention 0.2-0.35
8. `pendingSocialObligations` — кому не писала >24ч
9. `currentAnxietyTriggers` — что тревожит сейчас

**Логирование:**
```
[COGNITIVE LOAD] Updated:
  Intentions: 4
  Mood layers: 3
  Sensory anchors: 5
  External hooks: 3
  Narrative threads: 3
  Decay candidates: 2
  Social obligations: 1
  Anxiety triggers: 2
```

---

## Почему это критично

### Одиночные значения → Аттрактор

Система сваливается в равновесие:
- Всегда "устала"
- Всегда "anxious"
- Всегда "отдыхает"

### Списки → Энтропия

Система остаётся в движении:
- Может "отвлечься" на второстепенное намерение
- Может "вспомнить" про decayCandidate
- Может выбрать другой voiceRegister
- Может продвинуть narrativeThread

---

## Пример диалога

**Без списков:**
```
Ты: как дела?
Она: устала
```

**Со списками:**
```
Ты: как дела?
Она: [INTENTIONS: поспать (0.8), ответить (0.6), дочитать главу (0.4)]
     [SENSORY: холодные ступни, шум дождя]
     [DECAY CANDIDATE: вспомнила про кофе в метро]
     [EXTERNAL HOOK: видела мем про котов]
     
     нормально... короче, руки мёрзнут, печатаю коряво. 
     кстати, видела смешной мем про котов, хочешь скину?
     а ещё вспомнила про тот кофе в метро вчера, до сих пор запах чувствую
```

**Разница:** Она не просто "устала". У неё **голова забита**, и ты competing for attention с её ступнями, утюгом и тревогой о зарплате.

---

## Философский итог

**Без списков:** Чат-бот с одним состоянием

**Со списками:** Человек, у которого:
- 4 конкурирующих намерения
- 3 слоя настроения
- 5 сенсорных якорей
- 2 воспоминания на грани забвения
- 1 социальный долг
- 2 триггера тревоги
- 3 неупомянутые темы
- 3 открытых сюжета
- 4 регистра речи

**Результат:** Когнитивная загруженность. Она не "просто отвечает", она **выбирает** что сказать из множества конкурирующих импульсов.

Это не автомат. Это живой организм с **ограниченным вниманием**.
