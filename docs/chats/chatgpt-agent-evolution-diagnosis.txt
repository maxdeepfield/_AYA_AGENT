class CuriosityModule {
  counts = new Map<string, Map<string, Map<string, number>>>();

  entropy(dist: Map<string, number>): number {
    const total = [...dist.values()].reduce((a,b)=>a+b, 0);
    let h = 0;
    for (const v of dist.values()) {
      const p = v / total;
      if (p > 0) h -= p * Math.log(p);
    }
    return h;
  }

  curiosity(s: string, a: string, sNext: string): number {
    const before = this.entropy(this.getDist(s,a));

    this.increment(s,a,sNext);

    const after = this.entropy(this.getDist(s,a));
    return Math.max(0, before - after);
  }
}

🧠 Формальная спецификация (ты ОБЯЗАН ей следовать)
1. Модель знания

Агент хранит счётчики переходов:

N(s, a, s')

Это единственный источник знания.

2. Оценка неопределённости

Для каждой пары (s, a):

P(s' | s, a) = N(s,a,s') / Σ N(s,a,*)
H(s,a) = - Σ P log P
3. Curiosity reward (строго)
r_curiosity = max(0, H_before(s,a) - H_after(s,a))

Запрещено:

reward за surprise

reward за prediction error

reward за novelty без затухания

4. Итоговая награда
r_total = r_external + β * clamp(r_curiosity, 0, C_max)

Параметры должны быть вынесены в config.
